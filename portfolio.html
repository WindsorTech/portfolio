Portfolio

ANGULAR PERSONAL ORGANIZER is a powerful app programmed using the MEAN stack - MongoDB, Express, Angular and Node. It enables the users to create new tasks, by entering a task name and a due date, and save them into a database. It is also possible to view a list of all tasks ordered by date, where the users can tick the tasks as "completed", edit the information or delete any item. Click and try Angular Personal Organizer now!

DIGIBALL SOCCER is a project developed using the gaming framework Phaser. It is a challenging digital soccer game, where the player has 60 seconds to try to score as many goals as possible, while taking care to not concede any goals in the defence. The top 8 scores are always in the Leaderboard, play the game and try to be part of it!

GOTRIPPS.COM is a social travel website / mobile app that helps you to save time and money by quickly finding the best deals on flights, accommodation and activities anywhere on the planet! The app also has loads of useful travel information that is constantly updated by its users, and you can even create your own blog within GoTripps to tell the stories of your trips to the whole world!

INFOMEDIA SEARCH is a complete full-stack application that gives the user the power to search the world's top websites, getting relevant information about music, movies, trending news and even animated GIFs. I'm always improving this App, let me know what other search services you would like to see on InfoMedia!

SPOTITUBE is a cool app developed for a group project. Once users login with their Spotify account, the website finds matching YouTube videos for the songs in the users' playlist of choice.

ANGULAR NBA ALLSTARS is an app developed using AngularJS, and dedicated to all basketball fans out there! Here you are the coach, and you have the exclusive chance to pick your favorite NBA team of all time! What players will you choose? Play it now and build your own dream team!

WORLD CITIES HANGMAN is a fun game I developed using Javascript. Just like the iconic paper version of the game, the user has to try to guess which word - or city, in this case - the computer has chosen. Are you a traveler, do you know cities from around the world? Show us your skills playing World Cities Hangman! Have fun!

BAMAZON STORE is a command line Node app that simulates an Amazon-like storefront. It lists all the products available for sale, takes in orders from the customer, processes transactions and effectively depletes stock from the store's inventory, which is built using a MySQL database.

CRYSTALS COLLECTOR is a game developed using JQuery. The game starts generating a random number. The goal for the player is to match the random number by clicking on each crystal, which will add a specific amount of points to the total score. The value of each crystal is hidden from the user until he clicks on it, and the values change every time the game starts. Good luck!

RUSSIA 2018 TRIVIA GAME is an entertaining game dedicated to all soccer fans from around the world! How much do you know about the FIFA World Cup? Play the game and test your knowledge - the quicker you answer the questions, the more points you will get!

LIRI BOT is like iPhone's SIRI. However, while SIRI is a Speech Interpretation and Recognition Interface, LIRI is a Language Interpretation and Recognition Interface. It is a command line Node app that takes in parameters and gives back data to the user. LIRI's commands display useful information about any song, movies and more!

WORLD FACTS QUIZ is a game that will test your global knowledge. Developed using JQuery and made totally responsive for mobile devices, the game gives you 10 seconds to answer each one of the questions about world facts. At the end, the game presents the final results and evaluates your performance. Play it now and see how many questions you can get right, good luck!
